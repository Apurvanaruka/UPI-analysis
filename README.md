Ensure you have the following installed on your system:

# Installations
### create virtual enviorment
```
python -m venv env
```
### Activate virtual enviorment
```
source env/bin/activate
```
### Install dependencies
```
pip install -r requirements.txt
```
# Run app
### Change dir to src
```
cd src
```
